# PatientClaw

**患者诊后病情全自动跟踪统计系统 · 助力医生提升医术**

---

## 这是什么

PatientClaw 是一个基于 OpenClaw 构建的智能随访系统。

医生每天接诊几十名患者，但患者就诊后"一去不回"——药方是否有效、病情是否好转，医生完全不知道。PatientClaw 扮演医生的数字助理，自动承接随访工作：在正确的时间向患者发出专业问询，理解患者回复，整理成报告推送给医生。

**医生不需要改变任何工作习惯，系统在后台静默运行。**

---

## 核心功能

- **全自动随访**：就诊后第 3 / 7 / 14 天，自动生成并模拟发送随访消息给患者
- **患者回复模拟**：OpenClaw 自动模拟患者回复（含延迟、不回复等真实场景）
- **个性化问询**：结合每位患者的诊断和药方生成专属消息，不是通用模板
- **回复理解**：解析患者的自然语言回复，提取用药依从性、症状变化、不良反应
- **个人追踪报告**：为每位患者生成标准化的病情追踪档案
- **医生日报**：每天汇总所有患者情况，重点标注需关注的患者和用药方案效果，推送至医生飞书

**医生只需查看飞书日报，系统其余环节全部自动运行。**

---

## 快速了解

```
就诊 Day 0
  ↓
Day 3 → 飞书发随访消息 → 患者回复 → AI 解析记录
Day 7 → 同上
Day 14 → 同上 → 生成个人病情追踪报告
每天 → 生成医生综合日报 → 飞书推送给医生
```
