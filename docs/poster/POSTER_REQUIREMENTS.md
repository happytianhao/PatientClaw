## PatientClaw 海报设计需求

### 尺寸要求
- 物理尺寸：**0.8m × 2.0m**（A1 竖版），150 DPI
- 像素：4725px × 11811px
- 浏览器预览：**945px × 2362px**（1/5 比例）
- 打印时浏览器按 `@page { size: 842mm 2388mm }` 自动输出 1:1

### CSS 技术规范
- 不要用 CSS 变量 `--S`，不要用 `calc()`
- 所有 font-size / padding / margin / height 直接写固定 px 值
- 字体：body 14px，标题更大，PingFang SC / Microsoft YaHei / system-ui
- 风格：深色科技风，背景 `#080c14`，配色：绿/蓝/紫/青/橙/红
