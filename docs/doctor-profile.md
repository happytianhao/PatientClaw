## 从医理念

> **"每一次随访，都是对生命的守护。心血管疾病是慢性战争，医患并肩，才能赢得长远。"**

李晓峰教授从医二十余年，始终坚持"以患者为中心"的诊疗理念。他常说："技术是手段，人文是温度。一个好医生，不仅要有一双做手术的手，更要有一颗读懂患者的心。"

在高血压、冠心病、心衰等慢性心血管疾病管理中，李教授尤为重视院外随访与长期监测，倡导建立"诊前-诊中-诊后"一体化的全程管理模式。

## 专业方向

李晓峰教授长期深耕心血管疾病的临床诊疗与研究，在以下领域具有深厚造诣：

### 核心技术专长

| 领域 | 详情 |
|------|------|
| **冠心病介入治疗（PCI）** | 急性心肌梗死急诊 PCI、不稳定型心绞痛择期 PCI；复杂病变处理（分叉病变、左主干、慢性完全闭塞 CTO逆向开通），累计主刀PCI手术超过3000例 |
| **高血压综合管理** | 难治性高血压、继发性高血压筛查、原发性高血压合并靶器官损害（心、脑、肾）的综合管理；倡导家庭血压监测+远程随访 |
| **心律失常诊治** | 心房颤动节律/室率控制、射频消融术前评估与术后管理、起搏器/ICD/CRT 植入及长期随访 |
| **慢性心力衰竭** | HFrEF/HFpEF 指南导向药物治疗（GDMT）、心衰患者营养与康复指导、心室辅助装置评估 |

### 临床特色

- **精准用药**：根据基因多态性检测结果制定个体化抗栓/降压方案
- **介入微创**：擅长经桡动脉 PCI，术后即可下床，患者体验好
- **慢病随访**：建立患者微信群/飞书随访群，24小时响应患者问题
