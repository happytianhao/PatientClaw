PatientClaw

患者诊后病情全自动跟踪统计系统

医生不改变任何工作习惯，系统在后台静默运行——自动随访、理解回复、生成日报。

核心功能

📅

全自动随访

就诊后第 3 / 7 / 14 天自动触发，根据病情动态调整随访频率，无需人工干预。

💬

个性化问询

结合每位患者的诊断、用药、就诊天数生成专属消息，不是通用模板。

🤖

患者回复模拟

OpenClaw 自动模拟患者回复，含立即回复、延迟回复、不回复等真实场景。

🔍

回复智能解析

解析患者自然语言回复，提取用药依从性、症状变化、不良反应等结构化数据。

📋

病情追踪报告

为每位患者生成标准化病情追踪档案，记录完整随访时间线和趋势分析。

📊

医生每日日报

每天汇总所有患者情况，重点标注需关注患者和用药效果，推送至飞书。
