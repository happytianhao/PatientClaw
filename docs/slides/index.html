Prompt 工程：系统的真正核心
随访消息生成 Prompt 个性化生成
  • 1
    【用药情况】药有没有按时吃？有没有漏服或自行停药？
  • 2
    【不良反应】头晕、咳嗽、脚肿、皮疹等？
  • 3
    【自测指标】血压 / 心率 / 体重今天测了多少？
  • 4
    【核心症状】胸闷 / 头晕 / 气短 / 水肿有无改善？
  • 5
    【生活方式】饮食控盐、运动、烟酒情况？
高血压：血压值 + 服药 + 头晕头痛 心衰：每日体重 + 水肿 + 气短 心绞痛：胸痛频率 + 活动耐力 房颤：抗凝药 + 心悸 + 出血迹象
